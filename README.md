# XO game using Javafx
 <p>&emsp;• Nice looking User Interface.</p>
 <p>&emsp;• There's two levels of difficulty in single player mode.</p>
 <h3>The user can :</h3>
 <p>&emsp;• Play locally on the same machine in 1 VS 1 mode</p>
 <p>&emsp;• Play against other players on the same network in online mode.</p>
 <p>&emsp;• Record the games that you played in all modes (online games and local games are separated).</p>
 <p>&emsp;• Register and login to access online mode.</p>
 <p>&emsp;• Select who you want to play with from online players list.</p>
